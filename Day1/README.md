# 🚀 Node.js 30 Days - Backend Boilerplate

Dự án chuẩn bị nền tảng (Boilerplate) cho chuỗi 30 ngày học Node.js Backend. Được thiết lập với các tiêu chuẩn khắt khe nhất của dự án thực tế.

## ✨ Tính năng nổi bật
- Quản lý package siêu tốc với **pnpm**.
- Cú pháp hiện đại với **ES Modules (ESM)**.
- Quản lý cấu hình bảo mật với **dotenv**.
- Kỷ luật code nghiêm ngặt với **ESLint** & **Prettier**.
- Tự động bắt lỗi trước khi đẩy code với **Husky** & **lint-staged**.

## 🛠 Yêu cầu hệ thống (Prerequisites)
- [Node.js](https://nodejs.org/en) (Khuyến nghị bản LTS: v20 hoặc v22+)
- [pnpm](https://pnpm.io/installation) (v10+)
